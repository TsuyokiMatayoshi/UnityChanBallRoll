## 変数を定義
Rigidbodyコンポーネント力を加えるように編成するプロジェクト
計算を行うためには、まずは変数を用意

変数＝箱のようなもの
作ることを＝宣言＝変数の定義という　中に入るもの種類を変数の型

  ```Rigitbody = rb;```

変数の名前は何でもよい（日本語はダメ

変数は最初側の文で宣言する　定義したらほかの方は入れられない

## 値を代入
変数rbに値を入れる

  ```rb = GetComponent<Rigitbody>();```

コード上のイコールは右と左が正しいという意味ではない
右のものを左に代入することに使う　戻り値として返す

＝は代入の命令

## キーボードの入力を取得
```
Startメソッド｛ゲーム開始時に実行｝
Updateメソッド｛1秒間に数十回というすごい速さで実行Loop｝
```

移動とかゲーム中のキャラの動きはUpdateメソッド
値ゼロに返すゲーム開始処理などはStartメソッド

↑ゲーム開発ではよく使うメソッド（基本）

キーボードの入力に便利なコード
```
  float moveH = Input.GetAxis("Horizontal");
  float moveV = Input.GetAxis("Vertical");
```

HorizontalとVerticalの値を戻り値として受け取る
変数の宣言はfloatがその場でmoveHに少数を宣言している

変数の宣言と値の代入は
一つの行で簡略的にできる！！

## FixedUpdateの利用

最後の工夫

```
_Void Updateメソッドでは繰り返し処理がされているが、
フレームの実行回数はPCによって変わる　PCによって難易度も変わる

_void Fixedupdateメソッドを使うと　1秒間に週十回を
一定のスピードで実行を繰り返す。
```

Rigitbodyに力を加えるときは、FixedUpdate()を使いたい
